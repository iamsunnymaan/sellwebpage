üêõ Troubleshooting

If the menu doesn't open:

  • Check browser console for JavaScript errors (F12)
  • Verify that script.js is loaded
  • Ensure viewport width is ‚â§900px
  • Try hard refresh (Ctrl+Shift+R or Cmd+Shift+R)
  • Check if button has correct ID: "mobile-menu-toggle"
  • Check if nav has correct ID: "nav-box"

If dropdowns don't work:

  • Ensure .dropdown, .sub-dropdown classes are correct
  • Check that JavaScript adds/removes 'open' class
  • Verify CSS max-height transitions are working
